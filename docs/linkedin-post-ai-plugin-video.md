## LinkedIn Post — AI Plugin Generator (video demo)

---

Integrar um sistema novo ao seu monitoramento leva quanto tempo? Horas? Dias?

Com o orbit-core, leva segundos.

Nesse vídeo eu mostro o AI Plugin Generator em ação: descrevo uma API em texto livre e a IA gera tudo automaticamente:

- connector_spec.json (configuração completa)
- script Python pronto para deploy
- README com instruções de instalação

É só revisar, aprovar e os dados já começam a fluir.

Sem escrever código. Sem adapter manual. Sem ler documentação de API por horas.

Funciona com qualquer sistema que tenha uma API HTTP — Zabbix, CloudWatch, PagerDuty, APIs internas, IoT, o que for. Descreve em linguagem natural e a IA resolve.

O vídeo mostra o fluxo completo: da descrição até os dados aparecendo no dashboard. Menos de 2 minutos.

orbit-core é open source, self-hosted e gratuito.
